# Laozi Skill

这个仓库包含一个基于《道德经》与高可信学术材料整理的 `laozi-perspective` skill。

## 内容

- `laozi-perspective/SKILL.md`
  - 老子视角的主 skill 文档
  - 包含“文白夹杂，古意为骨，白话为解”的表达规则
  - 包含 10 组示例问答，用来展示这个 skill 在决策、领导、冲突、焦虑、产品、创作等场景下会如何发言
- `laozi-perspective/references/research/`
  - 6 份研究笔记，覆盖文本、表达、外部诠释、决策逻辑与时间线

## 设计原则

- 以《道德经》为核心文本
- 参考高可信二手资料做谨慎现代转译
- 区分“文本老子”与后世神仙化形象
- 风格上采用“文白夹杂，古意为骨，白话为解”

## 当前能力

- 用老子视角分析决策、领导、冲突、焦虑、创作与产品问题
- 识别过度控制、过度用力、过度争夺、过度堆加
- 给出偏减法、顺势、不争、知足的回应

## 示例问答说明

`laozi-perspective/SKILL.md` 中附带了 10 组示例问答。

这些示例不是单纯展示文风，而是用来说明这个 skill 的实际输出方式：

- 先下一句带古意的压缩判断
- 再用现代中文拆开意思
- 保持老子的减法、反转、不强作视角
- 避免滑成大段仿古文或玄学空话

如果后续要继续扩展，可以按同样格式增加更多场景化问答。

- 研究材料: `laozi-perspective/references/research/`
